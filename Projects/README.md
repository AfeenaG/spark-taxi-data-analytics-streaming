# Projects

This folder contains additional projects completed as part of the **Big Data Systems Design** course.

## Structure

Each project lives in its own subdirectory with:
- A `README.md` describing the problem, approach, and results.
- Source code, notebooks, or scripts relevant to the project.

## Adding a New Project

1. Create a new folder under `Projects/` with a descriptive name, e.g.:
   ```
   Projects/
   └── ProjectN-Short-Title/
       ├── README.md
       └── <source files>
   ```
